# Security Policy

Thank you for helping keep InternshipConnect secure!

Please report security vulnerabilities to security@internshipconnect.com
Do not report security issues through public GitHub issues.

We aim to respond within 48 hours and will work with you to address
any confirmed vulnerabilities as quickly as possible.
